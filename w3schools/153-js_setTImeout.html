Javascript setTimeout()

1) a test of execution order (keyword: Javascript event loop)

Code: 
    function clickButton1(e){
        console.log('Starting ... ');
        console.log('1');

        setTimeout(()=>{
            console.log('2');
        },1);

        console.log('3');
        console.log('4');

        setTimeout(()=>{
            console.log('5');
        },0);

        console.log('6');
        console.log('7');
    }
Output: 
    Starting ...
    1
    3
    4
    6
    7
    2
    5
Note

if remove console.log('6') and console.log('7'), output will be: 

    Starting ...
    1
    3
    4
    5
    2
