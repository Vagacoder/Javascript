CSS Pseudo-class

Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective! a:active MUST come after a:hover in the CSS definition in order to be effective!

A pseudo-class is used to define a special state of an element.

1) anchor tag <a> pseudo class

Go Google here
Go somewhere here
Go somewhere else here

2) simple tooltip hover

Mouse over me to show the tip

This is a tip

this is text after tip

3) :first-child pseudo class, matches a specified element that is the first child of another element.

this is first p

this is second p

this is third p
