JavaScript Regular Expressions

Syntax:

/pattern/modifiers;

e.g. var patt = /w3schools/i;

/w3schools/ is a regex

i is modifier (case-insensitive)

1) Search string using regex

Directly search using string in ""

Successfully found will return the starting index

No found will return -1

2) Replace string using regex

3) regex modifier: i (case-insensitive)

string is "Microsoft", regex is /micro/i

4) regex modifier: g (find all matches )

string is "Is this all there is?", regex is /is/g

5) regex modifier: m (multiline matching)

string is "\nIs th\nis it?", regex is /^is/m (multiline and beginning of line)
