JavaScript Regular Expressions

Syntax:

/pattern/modifiers;

e.g. var patt = /w3schools/i;

/w3schools/ is a regex

i is modifier (case-insensitive)

1) Search string using regex

Directly search using string in ""

Successfully found will return the starting index

No found will return -1

2) Replace string using regex

3) regex modifier: i (case-insensitive)

string is "Microsoft", regex is /micro/i

4) regex modifier: g (find all matches )

string is "Is this all there is?", regex is /is/g

5) regex modifier: m (multiline matching)

string is "\nIs th\nis it?", regex is /^is/m (multiline and beginning of line)

6) [abc] Find any ONE of the characters between the brackets

Find h in "Is this all there is?"

Find h a in "Is this all there is?"

Find 1, 2, 3, 4 in "1234567890"

7) | : Find any of the alternatives separated with |

8) \d : Find a digit

9) \s : Find a whitespace character

10) \b : Find a match at the beginning or at the end of a word

11) \uxxxx Find the Unicode character specified by the hexadecimal number xxxx

12) n+ : Matches any string that contains at least one n

13) n* : Matches any string that contains zero or more occurrences of n

14) n? : Matches any string that contains zero or one occurrences of n

15) test() method: searches a string for a regex, and returns true or false, depending on the result.

16) exec() method searches a string for a specified pattern, and returns the found text as an object. If no match is found, it returns an empty (null) object.
